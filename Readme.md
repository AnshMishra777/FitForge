# 🏋️ FITFORGE

FitForge is a modern full-stack fitness web application built to help users train smarter, track workouts, follow structured programs, and receive AI-powered nutrition guidance — all in one platform.

Designed with a clean dark-themed UI and performance-focused experience, FitForge combines workout management, exercise tracking, AI diet coaching, and customizable training routines into a single seamless application.

# ⚡ How FitForge Works

## 1. User Authentication
Users create an account or use the demo athlete account.

## 2. Dashboard Access
After login, users are redirected to the dashboard where they can:
- View current programs
- Start workouts
- Access AI nutrition
- Browse exercises

## 3. Workout Selection
Users can:
- Follow prebuilt programs
- Browse exercise library
- Open workout sessions
- Track progress live

## 4. Live Session Tracking
During workouts:
- Users complete sets
- Track rest times
- Monitor reps
- Progress through exercises

## 5. AI Nutrition Guidance
Users can ask nutrition-related questions and receive:
- Macro analysis
- Calorie calculations
- Meal recommendations
- Recovery advice

## 6. Custom Routine Creation
Users can build and save personalized workout splits tailored to their goals.
